# LFTools
Ulilities to easily mount/eject Didj and Leapster2 devices, while also being able to manage the filesystem.

# SH Version Commands:
## Mounting/Ejecting
-m: mount device

-l: lock device

-x: eject device

## Package Management
-d: get packages

-p: manage packages

## Miscellaneous
-c: emergency clear

-u: update didj

-h: help
# C Version Commands:
## Mounting/Ejecting
-m: mount device (requires device type)
-x: eject device (requires device type)

## Package Management
-p: package management

## Miscellaneous:
--update-didj: ...

-h: help
